Overview
This project analyses historical car sales and profitability across multiple dealers for SwiftAuto Traders. The objective was to transform transactional sales data into a structured executive dashboard providing clear insights into dealer performance, model profitability, and service-related risk indicators.
The solution combines Excel analytics and BI dashboarding (Cognos) to deliver both financial and operational intelligence.
